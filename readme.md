### 基本参数
* 机箱样式：NAS机箱
* 机箱型号：N8
* 机箱尺寸：220.4(宽)*244.5（深)*311.7mm(高)
* 主板支持：7"*7"(170*170MM)主板
* 硬盘位：8个3.5寸热插拔硬盘位
* 光驱位：无光驱位
* 电源支持：FLEX PSU
* 散热系统：2个12025风扇
* 扩展插槽：1个全高直插槽
* 前置端口：一个电源开关 一个USB3.0接口，一个电源工作指示灯，一个硬盘工作指示灯
* 产品材质：优质SGCC钢板
